# Releases

## Release 1 — 2026-07-17

나미 생일카페 프로젝트 첫 배포 버전. 2D 탑다운 픽셀 카페를 돌아다니며 오브젝트와 상호작용하는 생일 축하 웹페이지.

배포: https://namidew-valley.vercel.app

### 핵심 기능

| 오브젝트 | 모달 | 내용 |
|---|---|---|
| 액자 3개 | 사진 갤러리 | 방문객 사진 갤러리 |
| 카운터 | 영수증 | 영수증 스타일 축하 카드 |
| 방명록 나무 | 방명록 | 연도별 방명록 (리본 달기), Firestore 저장 |
| 케이크 | 촛불 불기 | 마이크로 실제 촛불 불기 (Web Audio 고주파 대역 판정) |
| 굿즈 전시대 | 굿즈 | 몬치치 굿즈 이미지 전시 (구매 아님, 구경 후 확인) |
| 전광판 | D-day | 생일까지 카운트다운, 생일 당일엔 축하 화면 |
| 카운터 직원 NPC | 대사 | 안내 대사 |
| 주크박스 | BGM 플레이어 | 트랙 선택 재생, 생일 당일엔 축하곡이 자동 기본값 |
| 포토부스 | 실시간 촬영 | 웹캠으로 직접 촬영 + 커스텀 프레임 합성 후 PNG 저장 |
| 가챠 | 포춘 뽑기 | 랜덤 축하 메시지 |
| 메뉴판 | 카페 메뉴 | 메뉴 목록 |
| 편지함 | 생일 편지 | 편지 + 컨페티 연출 |
| 방문객 NPC 3명 | 말풍선 | 배회하며 말풍선으로 대사 |

### 이번 릴리즈에서 작업한 것

**방명록 DB 마이그레이션**
- Supabase(무료 플랜 1주일 휴면 정책) → Firebase Firestore로 전환. 데이터/UI는 동일하게 유지
- `year` 필터 + `createdAt` 정렬 조합에 Firestore 복합 인덱스가 필요해 목록 조회가 조용히 실패하던 버그 발견 후, 정렬을 클라이언트 처리로 옮겨 인덱스 없이도 동작하도록 수정

**포토부스 개편**
- 캔버스로 그리던 가짜 포토카드 → `getUserMedia` 실시간 웹캠 촬영으로 변경
- 직접 제작한 프레임 이미지(`public/frames/`)를 촬영 사진 위에 합성해서 저장

**굿즈 매대 → 전시대**
- 장바구니/구매 로직 제거, 이미지 3장을 구경하고 확인 버튼으로 넘어가는 단순 상호작용으로 변경

**날짜 관련 로직**
- D-day 카운트다운과 기본 BGM을 "매년 반복되는 월/일" 기준으로 계산하도록 변경 — 생일이 지나면 자동으로 내년 생일로 카운트다운 재개, 기본 BGM도 생일 당일에만 축하곡으로 자동 전환

**캐릭터 선택 화면**
- 이모지 대신 실제 프로필 이미지(`male_profile.png` / `female_profile.png`)로 교체

**성능**
- 배경/굿즈 이미지 PNG → WebP 전환, BGM mp3 재인코딩으로 전체 에셋 용량 대폭 절감

**버그 수정**
- 새로고침해도 카페 상태로 바로 복귀하던 것을 막고, 항상 캐릭터 선택 화면부터 시작하도록 변경 (`zustand persist` 제거)
- 모바일에서 카메라 뷰포트 계산이 CSS `vh`와 JS `innerHeight` 불일치로 어긋나던 문제 수정
- 모바일에서 dpad가 게임 화면 하단과 겹쳐 이동 가능 영역이 가려 보이던 문제 수정 (레이아웃 정렬을 top-align으로 변경)
- 캐릭터 스프라이트 위치가 이중 보정으로 인해 타일 크기가 작을수록(모바일) 아래로 밀려 보이던 버그 수정
- 촛불 마이크 인식 개선: BGM이 마이크에 같이 잡히던 문제 해결(마이크 켜는 동안 BGM 일시정지), RMS 전체 음량 대신 고주파 대역 에너지 기반 판정으로 바꿔 "부는 소리"를 "말하는 소리"보다 더 잘 잡도록 수정

**기타**
- 파비콘을 생일 케이크 이모지로 변경
- Vercel 프로젝트명/도메인을 `namidew-valley`로 변경
